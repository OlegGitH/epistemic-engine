# Demo video script — target 2:45

## 0:00–0:18 — Problem

**Visual:** Title, then the Control Center.

“AI agents can recommend code changes and deployments faster than teams can verify the assumptions behind them. Epistemic Engine adds an evidence control plane between a recommendation and an action.”

## 0:18–0:40 — Product model

**Visual:** Account portfolio and one project.

“The Engine decomposes a recommendation into observable claims, binds typed evidence, exposes contradictions and unknowns, runs approval-bounded verification, applies deterministic policy and emits an immutable Decision Certificate.”

## 0:40–1:05 — Food Lens evidence

**Visual:** Food Lens salad example followed by the GitHub workflow.

“Food Lens is our transparent demo application. Raw images stay in the browser; only aggregate features and a content digest reach the API. Its pipeline produces executable test and privacy evidence rather than a hard-coded success fixture.”

## 1:05–1:40 — Four paths

**Visual:** Move through the four branch results in the dashboard.

“The same pipeline exercises four deliberate outcomes. Complete evidence is verified and allowed. Missing evidence creates critical unknowns. A PII-bearing log contradicts the privacy claim even after human approval. Finally, approved sandbox checks resolve two unknowns, while deployment remains conditional until a human approves it.”

## 1:40–2:05 — Investigation and report

**Visual:** Run debugger, then human certificate.

“Operators can inspect the evidence graph and exact policy path, or read the human report: proceed or do not proceed, the reasons, remaining conditions and the SHA-256 certificate digest. PostgreSQL restart tests prove the account, run, graph, certificate and report persist.”

## 2:05–2:32 — GPT-5.6 and Codex

**Visual:** Live-proof artifact and bounded patch output.

“GPT-5.6 runs through the Responses API and proposes a structured claim decomposition. It cannot approve an action or invent evidence. After explicit approval, the official Codex SDK may create one targeted verification test in a disposable checkout. Network access is disabled, non-test edits are rejected and the patch is never applied automatically.”

## 2:32–2:45 — Close

**Visual:** Dashboard outcome and tagline.

“Epistemic Engine turns agent confidence into inspectable evidence. Instrument once, verify before action, and send the result to any compatible engine. Evidence before autonomous action.”
